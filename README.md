## Build locally

```bash
# Install
pip install mkdocs mkdocs-material pymdown-extensions

# Convert vault → docs
python scripts/build_en.py
python scripts/regen_indexes_en.py

# Serve locally
mkdocs serve
# or
mkdocs build --strict
```
