# Coaching Methodology — Old Knowledge vs New Knowledge

The central organising framework of the 2026 coaching paradigm is the explicit distinction between **Old Knowledge** (classical coaching, built for wooden rackets and static bodies) and **New Knowledge** (neuro-motor science, built for the human nervous system as it actually works).

Every coaching intervention in this vault is grounded in one question: *does this instruction describe what the nervous system actually does — or what a robot arm would do?*

---

## The Paradigm Shift in One Sentence

Old Knowledge tells the player *how to swing*. New Knowledge creates an environment in which the player *discovers* the swing.

---

## Old Knowledge → New Knowledge: The Four Transitions

| Dimension | Old Knowledge | New Knowledge |
|---|---|---|
| **Power source** | Muscular effort; "swing harder" | Ground → kinetic chain → whip |
| **Stroke model** | One-unit, rigid arm, linear step | Segmented chain; time-lag sequencing |
| **Memory model** | "Muscle memory" through repetition | Motor engrams via myelinated neural pathways |
| **Coaching method** | Prescriptive cues; static basket drills | Constraints-led; representative learning design |
| **Core concept** | Static strength (planks, sit-ups) | Dynamic force transmitter (fascial slings) |
| **Error diagnosis** | Address error at the point of error | Trace error upstream: begin at the feet |
| **Mental model** | Willpower; mental toughness | CNS state management; implicit system trust |
| **Serve instruction** | "Big loops"; "complete shoulder turn" | 8-stage sequential model; arm relaxation |
